## 📖 About

**Nepflix** is a pixel-faithful, static clone of the Netflix UI reimagined as a dedicated platform for Nepali cinema. Every visual asset — logos, banners, and branding — was custom-crafted in **Adobe Photoshop** and **After Effects**. The full site runs on **HTML and CSS alone, with zero JavaScript**, relying exclusively on CSS animations, checkbox hacks, and native browser features for all interactivity.

The project features a curated library of real Nepali films, a cinematic intro sequence, a fully functional-looking sign-up flow, and a rich home browsing experience — all built from scratch.

## 🚀 Getting Started

No build tools, servers, or dependencies required. Just open the files in a browser.

### Clone the repository

```bash
git clone https://github.com/ayushshah-xo/Nepflix-HTML-and-CSS-.git
cd Nepflix-HTML-and-CSS-
```

### Open in your browser

```bash
# Option A — open directly (macOS)
open intro.html

# Option B — open directly (Linux)
xdg-open intro.html

# Option C — open directly (Windows)
start intro.html

# Option D — use a local dev server (recommended for videos to load)
npx serve .
# then visit http://localhost:3000/intro.html
```

### Recommended page flow

```
intro.html  →  open.html  →  signup.html  →  home.html  →  nepflix-watch.html
```

> **Note:** Videos in `assets/videos/` must be present locally for the TV section and intro to play. The watch page (`nepflix-watch.html`) links back from the home hero's Play and Trailer buttons.

## 📐 Pages Overview

### `intro.html` — Intro Screen
A full-screen black background with a centred video playback. A transparent `<a>` overlay spanning the entire viewport lets users click anywhere to skip to `open.html`.

### `open.html` — Landing Page
Mirrors the Netflix landing page structure:
- Sticky nav with logo and Sign In button
- Full-bleed hero with email CTA
- *Watch on your TV* section with embedded looping video
- *Download your shows* section with CSS-animated progress bar
- *Watch everywhere* section with floating device cards
- Five-question CSS accordion FAQ
- Multi-column footer with language selector

### `signup.html` — Sign-Up Page
Split-panel layout:
- **Left:** Full-height custom Nepali movie imagery
- **Right:** Form with floating labels for first name, last name, email, and password

### `home.html` — Browse / Home Page
- **Sidebar** with SVG icon nav (Home, Search, My List, Films, Trending, Classics, Profile)
- **Hero section** featuring *ऊनको स्वीटर* with IMDb badge, year, certificate, runtime, language, genre tags, synopsis, and Play / Trailer / Add-to-List buttons
- **Four movie rows** each with horizontally scrollable cards and hover overlays
- **Footer** with Nepali tagline and copyright

### `nepflix-watch.html` — Watch Page
Dedicated player page linked from the home hero's Play and Trailer buttons.

## 🐛 Known Issues

- Some movie poster paths in `home.html` reference `./assets/posters/` while the actual folder is `./assets/img/posters/` — update paths if posters aren't loading.
- The watch page video source needs to be configured locally (no hosted video stream is included).
